## 生存手册是什么？

由上海交通大学各个院系的毕业生们，分享他们在学习路上踩过的坑和填过的土。

包含出国、直博、保研、考研与就业/实习共五项，力求为 “如何在上海交通大学为申请国内、海外高校和入职优秀企业做出更全面的准备” 这一问题作出解答。

内容来源：[https://survivesjtu.gitbook.io/survivesjtumanual](https://survivesjtu.gitbook.io/survivesjtumanual)

二哥在此基础上，做了一些内容上的删改和批注，尤其是内容的格式上，倾注了大量的心血，甚至不惜修改了标点符号。因为我一直笃定，好的内容就应该有好的排版，只有这样，大家才真正能够把精华的内容吸收掉。

酒香也怕巷子深，我要做得的就是，让更多的人读到上交大这份生存手册，从而受益。
